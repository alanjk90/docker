Docker is a collection of below Ecosystem 1. Docker Client / Docker Cli -----> Tool that we are going to issue commands. 2. Docker Server/ Docker Daemon ---> Tools that is responsible for creating images, running container, etc 3. Docker Machine 4. DockerImage 5. Docker Hub 6. Docker Compose
